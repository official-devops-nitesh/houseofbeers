POS Manage Packages with Pricelist

Assign special price trough pricelist!!

Odoo POS Manage Packages with Pricelist module allows multiple quantities of a product to be bundled together in a package. The package can be then assigned a special price through a pricelist and sold in Odoo POS at that price(discounted or otherwise).

Need to create a different pricelist in Odoo?
Different Prices for Various Packages of the same product can be set using the same pricelist
All the available packages (if any) for a product are shown to the user in the Odoo POS when the product is added to the cart. The user can select the appropriate package and add it to the cart.
The product packages created for any product show up in the Odoo POS running session. The customer can choose the package and add it to the cart.
The Odoo POS order receipt also reflects the name of the product pack purchased by the customer.

Note: A bundle or a package can be made only for multiple quantities of the same product with this module.

Use Case Senario
Clearance Sale: End of season clearance sales allow the store owner to unload old products and make space for new incoming stock. Multiple packages of different quantities such as pack of 12, or package of two dozens, etc. can be created for any product and sold at a highly discounted price to the customers.
Bulk Discount: Groceries and edible products are often purchased in bulk by the customers. Providing a bundled product to the customer at some bulk discount allows for an upselling opportunity to the store owner.
Special pricelist on festivals and celebrations: It also helps you to sell slow moving products that are not popular or in demandn of the customers.Create multiple packages along with a fresh pricelist and provide a festive offer to your customers

Detailed Features List

Facilitates creating product bundles in Odoo

  • The POS Manage Packages with Pricelist Module facilitates the Odoo user in creating bundles of a product in Odoo POS
  • The module allows the user to arrange a product in packages of different quantities and sell them on Odoo POS.

Create and manage same product packages by pricelist

  • The Odoo user can create and manage packages of the same product using the pricelist.
  • Each package can be given a specific custom price (Discounted or otherwise) from pricelist.

Create multiple packages for discounted prices in Odoo

  • The multiple packages of the same products can be created and assigned different discounted Odoo POS prices using the pricelist.
  • The POS user can view the available package/bundle options for a product by clicking on the product in the POS session and adding them to cart in POS session.

View the product package name on the pricelist too in Odoo

  • The Odoo POS receipt also shows the name of the purchased product package
